# Fitness UI

This is the React frontend for the Fitness Microservices application.

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

The application will be available at http://localhost:3000

### Build
```bash
npm run build
```

### Preview
```bash
npm run preview
```

## Tech Stack
- React 18
- Vite
